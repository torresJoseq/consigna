¿Qué es HTML?

HTML es el lenguaje más usado para la construcción de páginas web.
HTML significa Hypertext Markup Language (lenguaje de marcas de hipertexto)
No es un lenguaje de programación, sino de marcas de hipertexto.
Llamamos a dichas marcas etiquetas. Estas le indican al navegador lo que debe mostrar en pantalla.
